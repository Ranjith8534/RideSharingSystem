# Ride Sharing System

## Overview

This project implements a class-based Ride Sharing System in C++ and Smalltalk for Assignment 5 of MSCS-632. The project demonstrates the three core object-oriented programming principles:

- Encapsulation
- Inheritance
- Polymorphism

## Project Files

- `RideSharingSystem.cpp` – C++ implementation
- `RideSharingSystem.st` – Smalltalk implementation

## Features

- Ride base class
- StandardRide and PremiumRide subclasses
- Driver and Rider classes
- Fare calculation
- Ride management
- Polymorphic ride processing

## OOP Concepts

### Encapsulation
Driver and Rider classes keep ride collections private and provide methods to manage them.

### Inheritance
StandardRide and PremiumRide inherit from the Ride base class.

### Polymorphism
Different ride types are stored in a common collection and processed through the same interface.
